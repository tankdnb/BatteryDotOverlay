Battery Dot Overlay

Quick start

1. Unpack the release archive into any folder.
2. Make sure SteamVR is installed, running, and the headset is connected.
3. If you use a PICO headset, keep PICO Connect running because battery fallback is read from its local logs when OpenVR does not expose HMD battery.
4. Start BatteryDotOverlay.exe.
5. The app will show a small semi-transparent yellow dot in the upper-right area of view when HMD battery is below 100%.
6. If battery is 100%, the dot stays hidden.
7. To stop the app, close the console window or press Ctrl+C in the console.

Important

- This release is self-contained: no .NET installation is required.
- SteamVR/OpenVR runtime is still required on the test PC because the overlay works through SteamVR.
- On PICO headsets the app can read headset battery from recent PICO Connect battery events if the OpenVR driver does not expose battery percentage.
- Settings are stored in config\indicator.settings.json.
- Russian quick-start guide: START-HERE-RU.txt
- Russian configuration guide: CONFIGURE-OVERLAY-RU.txt
