Battery Dot Overlay

Quick start

1. Extract the release archive to any folder.
2. Make sure SteamVR is installed, running, and your headset is connected.
3. For Android headsets, USB debugging plus a working `adb` connection gives the most reliable charging-state detection.
4. If multiple Android devices are connected, set `battery.adb_device_serial` in `config\indicator.settings.json`.
5. Start `BatteryDotOverlay.exe`.

What is new in this build

- duplicate launches are blocked more safely;
- the app writes a persistent log file;
- `--validate-config` checks the config without starting the overlay;
- startup errors are more descriptive.

Useful commands

- `BatteryDotOverlay.exe --show-log-path`
- `BatteryDotOverlay.exe --validate-config`
- `BatteryDotOverlay.exe --diagnose-battery`

Notes

- The release is self-contained and does not require a separate .NET installation.
- SteamVR/OpenVR is still required because this is an overlay application.
- Main config file: `config\indicator.settings.json`
- Parameter reference: `CONFIGURE-OVERLAY.txt`
