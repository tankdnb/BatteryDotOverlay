Battery Dot Overlay
Подробная инструкция по настройке

Основной файл конфигурации:
config\indicator.settings.json

Важно

- Файл конфигурации поддерживает комментарии.
- После изменения файла нужно перезапустить приложение.
- Если не уверены, меняйте параметры по одному.

Раздел overlay

- overlay.overlay_key
  Уникальный идентификатор overlay в OpenVR/SteamVR.
  Если хотите запускать независимые копии, у каждой должен быть свой ключ.

- overlay.overlay_name
  Человекочитаемое имя overlay.

- overlay.width_in_meters
  Размер точки в VR-пространстве.

- overlay.distance_meters
  Расстояние точки от глаз.

- overlay.horizontal_offset_meters
  Смещение по горизонтали.
  Положительное значение двигает точку вправо.

- overlay.vertical_offset_meters
  Смещение по вертикали.
  Положительное значение двигает точку вверх.

- overlay.sort_order
  Порядок отрисовки overlay.

Раздел visual

- visual.color_hex
  Цвет точки в формате #RRGGBB.

- visual.opacity
  Прозрачность точки.
  0.0 = полностью прозрачная.
  1.0 = полностью непрозрачная.

- visual.texture_size_pixels
  Размер внутренней текстуры круга.

- visual.circle_fill_ratio
  Насколько большой круг рисуется внутри прозрачной текстуры.

- visual.edge_feather_pixels
  Мягкость края круга.

Раздел behavior

- behavior.blink_frequency_hz
  Частота мигания.
  1.0 = один цикл мигания в секунду.
  0.0 = без мигания, точка просто постоянно видна.

- behavior.blink_below_percent
  Порог заряда, ниже которого индикатор может включиться.

- behavior.blink_only_when_not_charging
  Учитывать ли зарядку шлема.
  По умолчанию: true.

- behavior.battery_poll_interval_ms
  Как часто приложение обновляет батарею и статус зарядки.

- behavior.status_log_interval_seconds
  Как часто приложение пишет статус в консоль и лог.

Раздел battery

- battery.preferred_device_index
  Принудительный индекс tracked device для чтения через OpenVR.
  -1 = автоматический режим.

- battery.log_diagnostics_on_startup
  Печатать ли подробную диагностику при старте.

- battery.log_diagnostics_when_unavailable
  Печатать ли дополнительную диагностику, если батарея не была получена.

- battery.enable_adb_fallback
  Включить ли ADB fallback.
  Это основной универсальный fallback для Android-шлемов, если OpenVR не помогает.

- battery.adb_path
  Путь к adb.exe или к папке, где лежит adb.exe.
  Пустая строка = автопоиск.

- battery.adb_device_serial
  Serial Android-устройства.
  Нужен, если к ПК подключено несколько Android-устройств.

- battery.adb_command_timeout_ms
  Таймаут ADB-команды.

- battery.enable_pico_connect_log_fallback
  Разрешить ли запасной fallback через логи PICO Connect.

- battery.pico_connect_logs_path
  Пользовательский путь к логам PICO Connect.

- battery.pico_connect_event_max_age_seconds
  Максимальный возраст событий из PICO Connect.

- battery.pico_connect_log_tail_bytes
  Сколько данных читать с конца логов PICO Connect.

Раздел runtime

- runtime.loop_delay_ms
  Внутренняя задержка рабочего цикла приложения.

- runtime.enable_file_logging
  Включить ли логирование в файл.
  По умолчанию: true.

- runtime.logs_directory
  Папка для логов.
  Можно указывать относительный путь, тогда он считается от папки exe.

- runtime.max_log_files
  Сколько последних логов хранить.

- runtime.enforce_single_instance
  Защита от повторного запуска второй копии приложения с тем же overlay_key.
  По умолчанию: true.

Новые сервисные команды

- `BatteryDotOverlay.exe --validate-config`
  Проверяет конфиг и показывает итоговые пути.

- `BatteryDotOverlay.exe --show-log-path`
  Показывает путь к активному лог-файлу.

- `BatteryDotOverlay.exe --diagnose-battery`
  Показывает диагностику батареи и зарядки.

Рекомендуемая настройка под обычный сценарий

- behavior.blink_frequency_hz = 1.0
- behavior.blink_below_percent = 90.0
- behavior.blink_only_when_not_charging = true
- battery.enable_adb_fallback = true
- runtime.enable_file_logging = true
- runtime.enforce_single_instance = true
- visual.color_hex = #FFD54A
- visual.opacity = 0.55
