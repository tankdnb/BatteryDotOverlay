Battery Dot Overlay

Быстрый старт

1. Распакуйте архив релиза в любую папку.
2. Убедитесь, что SteamVR установлен, запущен, и шлем подключён.
3. Если у вас Android-шлем, для лучшего определения зарядки желательно включить USB-отладку и проверить, что `adb devices` видит шлем.
4. Если к ПК подключено несколько Android-устройств, укажите serial шлема в `config\indicator.settings.json` через `battery.adb_device_serial`.
5. Запустите `BatteryDotOverlay.exe`.

Что делает приложение по умолчанию

- Показывает мигающую жёлтую точку, только если заряд шлема ниже заданного порога.
- По умолчанию дополнительно проверяет, идёт ли зарядка.
- Если шлем заряжается, индикатор скрывается.
- Если статус зарядки определить не удалось, приложение сохраняет старое поведение и ориентируется только на процент батареи.

Что нового в этой сборке

- Есть защита от повторного запуска второй копии.
- Появился лог-файл.
- Появилась команда `--validate-config`.
- Ошибки старта теперь объясняются подробнее.

Полезные команды

- Показать путь к активному лог-файлу:
  `BatteryDotOverlay.exe --show-log-path`

- Проверить конфиг без запуска overlay:
  `BatteryDotOverlay.exe --validate-config`

- Вывести диагностику батареи:
  `BatteryDotOverlay.exe --diagnose-battery`

Важно

- Сборка автономная: установка .NET не требуется.
- Для работы overlay всё равно нужен SteamVR/OpenVR runtime.
- Главный файл настроек: `config\indicator.settings.json`
- Подробное описание параметров: `CONFIGURE-OVERLAY-RU.txt`

Как остановить приложение

- Закройте окно консоли.
- Или нажмите `Ctrl+C`.
